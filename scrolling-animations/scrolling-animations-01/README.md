# Scrolling-Animations: "Scrolling Image Animation"

An Image that rotates on-scroll.

## Keynotes

+ Use an image

+ Use the window to determine how far your scroll down/up

+ `window.pageYOffset`

+ `[image].style.transform = `rotate(${scroll}deg)`;`
